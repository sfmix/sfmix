---

The classic backbone weathermap: every switch in the SFMIX fabric and every
physical link between them, coloured by how much traffic each one is carrying
right now. Each parallel fiber is drawn as its own strand, and each strand is
split in the middle so you can see both directions independently — chevrons
point the way the bits are flowing. Hover or tap any link for the numbers.
