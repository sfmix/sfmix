The OG backbone weathermap: every switch in the SFMIX fabric and every physical
link between them, color-coded by how cooked each one is rn. Every parallel
fiber gets its own strand, split down the middle so you can see both directions
separately — the chevrons show which way the bits are going. Hover or tap any
link for the receipts.
