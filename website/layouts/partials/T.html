{{- /* T — gettext-style translation helper. The English source string IS the key (the msgid), exactly like gettext / Django's {% translate %}. There is no en.toml: English is the source, so authors just write plain English in templates. Translators fill in i18n/.toml files whose keys are the English sentences and whose values are the translations (the .po "msgid -> msgstr" model). Any string with no entry for the current language falls back to the English source. Usage: {{ partial "T.html" "Connect to SFMIX" }} HTML: {{ partial "T.html" "Contact us." | safeHTML }} */ -}} {{- /* Return the raw (unescaped) string so the caller controls escaping. Emitting the value with {{ . }} here would HTML-escape it inside this template's html/template context, which breaks HTML-bearing translations (e.g. a string containing ): the markup gets escaped before the call site's | safeHTML can mark it safe, leaving <a> in the output. With `return`, plain-text call sites still escape normally and HTML call sites that pipe through | safeHTML render correctly. */ -}} {{- $msgid := . -}} {{- return (or (i18n $msgid) $msgid) -}}
